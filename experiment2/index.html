Description

This project offers a generative, living impression of this inspiring image:

[source](https://www.google.com/url?sa=i&url=https%3A%2F%2Fstock.adobe.com%2Fsearch%3Fk%3Dpsychedelic%2Bforest&psig=AOvVaw0Z39Oz1-hmobwXH7NVsr69&ust=1713215822168000&source=images&cd=vfe&opi=89978449&ved=0CBIQjRxqFwoTCJDr1b7QwoUDFQAAAAAdAAAAABAE)

Technical

Relation to inspring image:

  • Color: The sky, grass, mushrooms, and river colors were sampled from the inspiring image using a website eyedropper tool.
  • Primitives: The sky is rendered with `rectangle` primitives.
  • Polygons: The background river is a single randomly polygon using `vertex` calls.
  • Composition: Different random seed values lead to differently shaped mushroom caps, stems and stars.
  • Life: The mushroom caps spin along as the stems but those are harder to tell since there are no strokes. The river also moves as if it's breathign I wanted to warp it more but I couldn't figure out how to curve it as such so I was stuck with a straight lined river. The stars also twinkle over time.
