Experiment 4b - Infinite World: Rainbow Land

Description

With this iteration I wanted to mess with the sporadic color change. Originall the idea was to make a dance floor, but then I realized if I made the whole floor a dance floor using this method then the user's world key input wouldn't seem to matter as much. So a new inspiration came to be when I remembered Rainbow World from Mario Kart. I thought it'd be a good idea to make the land for Rainbow World and the gaps would be pitch black since it's up in space. Follwoing that inspiration I decided to make the users mouse clicking spawn in a low quailty version of Mario.

Technical

To achieve this look I had created an array of colors inspired by a dance floor and then a rainbow. I then used this array in a random choice selction for the fill line of the tiles. In the if statement I then made any other tile not in the has equation be black to represent the empty void. To create "Mario" I made him out of rectangles of varrying sizes with specific colors to give the ressemblance of Mario. Then I used the same has equation for the tiles for the placement of Mario so that he could only be placed on the land of Rainbow World.

Reflection

This one was a lot quicker, at least for the land. The only thing that took long was making the rectangles look like Mario and stay in the boundaries. The use of the "glitch" was fun to implement intot he creation of the world.
