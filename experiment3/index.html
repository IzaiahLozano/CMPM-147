Experiment 3 - Alternate Worlds

Description

There are two parts to this experiment. For the overworld I was looking to create a biodiverse landscape surrounded by water. Something very similar to the example shown. For the Underworld I wanted to make a dungeon lair with multiple caves and a random chance that a treaure chest would appear. With every click of the reseed button a new generation of the worlds will be created providing a unique landscape with every click. The users could also use a text box to edit the canvas directly and change the map to their liking.

Technical

A lot of what I did came down to multiple for loops and if statements. I need to fill the canvas with a mix of characters like '_' or '$'. These charcaters woudl then help me when I would iterate over the canvas and checking the characters. If the specific character was found a tile would be placed in the exact same x,y coordinate as in the text box but in the canvas. For the worm I used an ellipses and another for loop to draw the worm across the land. For the chest chance I needed a simple math equation of probability and added that in whent he canvas was generated.
